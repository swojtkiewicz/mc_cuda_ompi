Template to save array of arbitrary type to file.
Array can be 1D, 2D or 3D.

If array is 3D, each z dimension is saved in a different file (filaname_zDim=*.txt), where *=1, 2 or 3.

Parameters
datapointer to an array of arbitrary type, the array is an linear array
filenamefile name to save the array
array_sizedefines dimension of the array 1D, 2D or 3D
xDimsize of the array in x dimension
yDimsize of the array in y dimension
zDimsize of the array in z dimension
Exceptions
throwsall exceptions
  • x dimension in file is along rows
  • y dimension in file is along columns
  • if array is 2D, the zDim is ignored
  • if array is 1D, the yDim and zDim are ignored
  • if array_size != {1,2,3} this function does nothing

◆ saveVoxels()

void saveVoxels ( string  voxels_filename,
float *  voxels,
Settings *  settings 
)

Save voxelized structure with weights in each voxel to file.

Parameters
voxelspointer to array holding voxels structure
settingspointer to settings structure
voxel_parameterspointer to voxel parameters structure
Exceptions
throwsall exceptions

Binary output file format:

  1. 1 x unsigned int
    - voxel structure x dimension (x_dim)
  2. 1 x unsigned int
    - voxel structure y dimension (y_dim)
  3. 1 x unsigned int
    - voxel structure z dimension (z_dim)
  4. x_dimmension * y_dimmension * z_dimmension x float32
    - weights of voxels

To get weight of voxel in position

(ind_x, ind_y, ind_z)

use formula:

ind_x + x_dim*ind_y + x_dim*y_dim*ind_z

where voxels positions starts with 0
